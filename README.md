# TaskFlow Next.js App

A simple task management application built with Next.js.

## 🚀 Features

- User Registration (localStorage)
- User Login (validation)
- Form handling with React hooks
- Data fetching using fake API
- Clean and responsive UI

## 🛠️ Tech Stack

- Next.js
- React
- JavaScript
- CSS Modules

## 📚 What I Learned

- useState and form handling
- useEffect and data fetching
- Authentication flow (register & login)
- Working with localStorage

## 📦 Installation

```bash
npm install
npm run dev
